# KeenKeeper

> Your personal shelf of meaningful connections. Browse, tend, and nurture the relationships that matter most.

## Description

KeenKeeper is a relationship management app that helps you stay connected with the people who matter in your life. Set contact goals, track interactions, and get reminded when a friendship needs attention — before it fades.

## Technologies Used

| Technology | Purpose |
|---|---|
| React.js | UI framework |
| React Router DOM | Client-side routing |
| Recharts | Pie chart for analytics |
| react-hot-toast | Toast notifications |
| Vite | Build tool & dev server |
| Geist | Typography |

## Key Features

1. **Friend Cards with Status Tracking** — View all your friends in a 4-column grid with color-coded status badges (On-Track, Almost Due, Overdue).

2. **Quick Check-In Logging** — Log a Call, Text, or Video from any friend's detail page. It's instantly added to the Timeline with a toast notification.

3. **Friendship Analytics** — A donut chart on the Stats page shows your interaction breakdown by type.

## Getting Started

```bash
npm install
npm run dev
```
